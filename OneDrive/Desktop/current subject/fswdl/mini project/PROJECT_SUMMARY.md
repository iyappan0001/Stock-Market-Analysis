## 🚀 Ready to Run!

### Quick Start (3 Steps)

#### Step 1: Install Backend Dependencies

```powershell
cd backend
npm install
```

#### Step 2: Install Frontend Dependencies

```powershell
cd frontend
npm install
```

#### Step 3: Start Both Servers

```powershell
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend
cd frontend
npm start
```

The application will open at: **http://localhost:3000**

## 💾 Database Setup

### MongoDB Required!

#### Option 1: Local MongoDB

1. Download from: https://www.mongodb.com/try/download/community
2. Install and start MongoDB
3. Use default connection: `mongodb://localhost:27017/stock_market`

#### Option 2: MongoDB Atlas (Cloud)

1. Create account: https://www.mongodb.com/cloud/atlas
2. Create free cluster
3. Get connection string
4. Update `backend/.env`:
   ```
   MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/stock_market
   ```

## 🐛 Common Issues & Solutions

| Issue                        | Solution                                       |
| ---------------------------- | ---------------------------------------------- |
| "Cannot connect to database" | Install MongoDB or use Atlas connection string |
| "Port 5000 in use"           | Kill process or change PORT in .env            |
| "Frontend won't load"        | Ensure backend is running first                |
| "CORS errors"                | Check backend is running on port 5000          |
